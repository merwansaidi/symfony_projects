# Intro Symfony

[Keynotes et PPT](keynotes-ppt)

## Récupération du projet
1. `git clone https://github.com/yoanncoualan/IPSSI_201920_AWBD_Symfony_intro.git`
2. `cd IPSSI_201920_Symfony_intro`
3. `composer install`
4. Mettre à jour le fichier `.env`
